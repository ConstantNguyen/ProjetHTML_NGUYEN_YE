À propos

Ce site web a été réalisé dans le cadre de notre projet en programmation web. Il a pour objectif de présenter les cours et formations proposés, ainsi que les enseignants qui les dispensent, à travers une interface moderne et facile à utiliser. Le site a été développé en utilisant HTML, CSS et JavaScript. Ces technologies permettent de créer des pages structurées, interactives et adaptées à tous les types d’écrans. Une attention particulière a été portée à l’expérience utilisateur, avec une navigation fluide et des contenus clairs. La page Cours et formations permet de consulter les différents cours sous forme de cartes interactives. Chaque carte affiche les informations principales du cours et propose un bouton En savoir plus qui ouvre une fenêtre détaillée avec la description complète et les informations sur l’intervenant. Les données sont chargées à partir d’un fichier JSON, ce qui facilite la mise à jour du contenu. Le site intègre également des fonctionnalités supplémentaires comme des filtres de recherche, un menu responsive et un assistant de discussion pour guider les utilisateurs. Ce projet permet de mettre en pratique les bases du développement web et de comprendre comment créer un site dynamique, structuré et évolutif.
